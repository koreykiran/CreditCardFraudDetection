Technology: Python

Software and Hardware requirements

Software requirements:
• Windows 7/8/10.
• Python version 3.6.
• Anaconda version 3.
• Jupyter notebook/ipython.
• Python libraries(tenserflow, numpy, pandas, sklearn).

Hardware requirements:
• RAM: 8 GB.
• Processor I5 and above.
• Minimum 10 GB Hard disk space.

Steps to run the project:

Step 1: Install the required softwares

Step 3: Download the zip file, extract it and goto the directroy and open command prompt.

Step 3: run command -> jupyter notebook in the directory.(New tab opens in Browser, click on CreditCardFraudDetection.ipynb file.)

Step 4: Make sure that the all the csv files are present in the same folder

Step 5: To run all the cells go to kernel tab and select restart and run all
